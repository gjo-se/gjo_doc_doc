# Escaping

You can all escaping with a Backslash: \

```markdown
\\   backslash
\`   backtick
\*   asterisk
\_   underscore
\{\}  curly braces
\[\]  square brackets
\(\)  parentheses
\#   hash mark
\+   plus sign
\-   minus sign (hyphen)
\.   dot
\!   exclamation mark
\`not codeblock`
\*not italics*
\**not bold**
\[not link](http://www.google.com)
```
